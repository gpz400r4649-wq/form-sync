# フォームシンク

iPad Safariで使う、体育のフォーム比較用Webアプリです。お手本動画と生徒動画はブラウザ内でのみ読み込まれ、外部へ送信されません。

アプリ本体は `index.html` だけで構成されています。CSS、JavaScript、画像、フォント、外部ライブラリへの依存はありません。

## 使い方

1. GitHub PagesのURLをiPadのSafariで開きます。
2. お手本動画と生徒動画をそれぞれ選択します。
3. 共通の再生ボタン、シークバー、コマ送り、再生速度を使って比較します。
4. 動きの開始が合わない場合は「生徒動画の同期調整」で0.1秒ずつ調整します。

## プライバシー

- 動画は `URL.createObjectURL()` で端末内から直接読み込みます。
- 動画を外部へ送る通信処理はありません。
- 外部CDN、外部フォント、外部画像、外部ライブラリを使用しません。
- 動画形式は `.gitignore` で除外し、誤ってGitHubへ登録しにくくしています。

## ローカルでの確認

このフォルダーで `python -m http.server 8000` などを実行し、`http://localhost:8000` を開いてください。
